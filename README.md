# 🏫 SchoolFix

**School Maintenance Management System**

A modern web application that allows students and school community members to easily report broken or damaged school materials and facilities.

---

## 🚀 Project Overview

**SchoolFix** is a full-stack web application designed to streamline the maintenance reporting process in schools. Students, teachers, and staff can quickly report issues such as broken desks, faulty lights, leaking taps, or any damaged school property through an intuitive interface.

Admins can view all submitted reports, track their status, assign tasks to maintenance staff, and provide instructions for repairs — making school maintenance faster, organized, and more efficient.

---

## ✨ Features

- User Registration & Login for Students and Staff
- Secure Password Hashing (server-side)
- Session-based Authentication
- Role-Based Access Control (Admin & User roles)
- Easy Issue Reporting Form (with photo upload support)
- Real-time Report Dashboard for Admins
- Admin can update report status and add repair instructions
- View all maintenance reports with details
- Responsive & Modern User Interface
- MySQL Database Integration
- RESTful PHP Backend API
