Project Overview

This is a full-stack web application built for practical learning and real-world understanding of how frontend and backend systems work together. The project uses a simple but effective technology stack, focusing on clarity, structure, and best practices rather than heavy frameworks.

The application includes a complete authentication and authorization system, role-based access control, database integration, and secure handling of user data.

---

Features

* User registration and login
* Secure password handling on the server
* Authentication using sessions
* Authorization with user roles (Admin and User)
* Role-based access control for protected routes
* MySQL database integration
* PHP backend API
* Frontend and backend communication using HTTP requests
* Recommended image upload handling (files stored on server, file paths stored in database)

---

Technology Stack

Frontend

* HTML
* CSS
* Vanilla JavaScript

Backend

* PHP
* MySQL

Server Environment

* XAMPP (Apache and MySQL)
